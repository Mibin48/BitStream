# BitStream

BitStream is a high-fidelity, Neobrutalist messaging and collaboration platform built with React, Tailwind CSS, and Framer Motion.

## Features

- **Real-time Messaging**: Threaded conversations with rich media support.
- **HD Video Calls**: Crystal clear 1080p video conferencing.
- **Smart Scheduling**: AI-powered calendar integration.
- **Workspaces**: Organize teams and projects into dedicated isolated spaces.
- **Accessibility**: ARIA labels, focus states, and reduced motion support.

## Tech Stack

- **Frontend**: React, Vite, TypeScript
- **Styling**: Tailwind CSS (Neobrutalist design system)
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Mibin48/BitStream.git
   ```
2. Navigate to the frontend directory:
   ```bash
   cd BitStream/frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
