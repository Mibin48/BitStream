# ⚡ BitStream

**BitStream** is a high-fidelity, Soft Neobrutalist collaboration platform engineered for high-velocity teams. It blends a bold, geometric aesthetic with sophisticated glassmorphism and physics-based interactions to create a workspace that feels tactile, premium, and alive.

## ✨ Premium Features

### 🏢 Intelligent Workspaces
*   **Contextual Hubs**: Isolated environments for teams, projects, and departments.
*   **Smart Sidebar**: A semi-collapsed, high-contrast navigation system designed for maximum focus.

### 🎥 Spatial Communication
*   **Video Calls 2.0**: A floating, glassmorphic HUD with spatial controls and real-time recording indicators.
*   **Live Emoji Bursts**: Physics-based reaction system. Fire off reactions that float with randomized trajectories and rotations for immersive social presence.
*   **Voice Rooms**: Low-friction audio channels with animated portal transitions and interactive participation modes.

### 📂 File Management & UX
*   **Quick Peek**: Spacebar-triggered high-fidelity file previews. Preview documents, images, and archives without leaving your current view.
*   **Soft Neobrutalism**: A refined design language using high-contrast tokens (`#8b8abc`, `#c5f06c`) and deep-layered shadows.

### 📅 Advanced Productivity
*   **Tactile Calendar**: Interactive event resizing via drag-handles and multi-view (Daily/Weekly/Monthly) synchronization.
*   **Command Palette**: Global search and navigation via a semantic `Cmd+K` interface.

## 🛠️ Tech Stack

*   **Core**: [React 18](https://reactjs.org/) + [TypeScript](https://www.typescriptlang.org/)
*   **Build Tool**: [Vite](https://vitejs.dev/)
*   **Motion**: [Framer Motion](https://www.framer.com/motion/) (Physics-based animations, layout transitions)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/) (Custom Design System)
*   **Iconography**: [Lucide React](https://lucide.dev/) (Refined stroke-width: 2.5)

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Mibin48/BitStream.git
   ```

2. **Install Dependencies**
   ```bash
   cd BitStream/frontend
   npm install
   ```

3. **Launch Development Environment**
   ```bash
   npm run dev
   ```

## 🎨 Design Philosophy

BitStream follows a **"Functional Neobrutalist"** approach. We prioritize:
- **Visual Weight**: Thick borders and high-contrast colors to define hierarchy.
- **Glassmorphism**: 40px+ blur layers for depth and focus.
- **Micro-Interactions**: Scale-pulsing, layout-snapping, and spring-based physics on every interactive element.
